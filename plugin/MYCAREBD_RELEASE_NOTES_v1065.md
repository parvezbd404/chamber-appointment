# MyCareBD / Chamber Appointment System v1.0.65

## Automatic available-serial loading

- On the patient booking page, selecting both a doctor and an appointment date now automatically loads the available serial cards.
- The extra **Show Available Serials** button is removed from the patient booking screen.
- Changing the doctor, patient, or date refreshes the serial list safely.
- If the selected patient already has an active appointment, the existing blocking warning still appears and serials do not open.
