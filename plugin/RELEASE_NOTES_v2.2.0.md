# Chamber Appointment System 2.2.0

## Patient portal UX refresh

- Introduces an original, clinically oriented patient-portal design inspired by common best practices in major patient portals.
- Adds a task-first dashboard with quick cards for booking, visits, messages, and family access.
- Promotes the next appointment as the central dashboard item.
- Applies a calm clinical palette, clearer typography, stronger hierarchy, and accessible card patterns.
- Retains desktop top navigation and the mobile bottom navigation introduced in 2.1.9.
- Preserves all appointment, OTP, waiting-list, message, family-profile, notification, and security workflows.
