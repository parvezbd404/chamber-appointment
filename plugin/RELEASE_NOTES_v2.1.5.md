# Chamber Appointment System 2.1.5

- Added configurable OTP length from 4 to 8 digits under OTP Settings. The default remains 6 digits.
- Updated web and mobile OTP generation and verification to honor the configured length.
- Deactivated doctors/chambers now appear with a pale-red row and red Inactive badge in the backend.
- Deactivated patients use the same visual treatment in the Patients list.
