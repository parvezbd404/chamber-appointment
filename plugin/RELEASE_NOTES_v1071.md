# Chamber Appointment System v1.0.71

## Automatic available serial loading repair

- Available serials now load reliably after a date is selected, including mobile browser date pickers.
- Supports input, change and blur event fallbacks while preventing duplicate requests.
- If the automatic request cannot finish, a visible “Show available serials” retry control appears with an error message.
- No change to appointment rules, confirmation layout, OTP, Diabetes Care or backend workflows.
