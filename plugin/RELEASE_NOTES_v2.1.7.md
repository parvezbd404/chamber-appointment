# Chamber Appointment System 2.1.7

## Frontend new-profile fix

- Address is no longer required when creating a new patient profile after OTP verification.
- Minimum required fields remain verified mobile number, full name, valid age, and gender.
- Address, email, blood group, and city can be completed later from the patient profile.
- Mobile API registration now follows the same minimum-field rule.
- Existing address values and profile editing remain supported.
