# Chamber Appointment System 2.0.9

- Reorganized **Appointments** into two tabs: **All Appointments** and **Print Patient List**.
- Kept date, doctor, status, search, filter, reset, CSV export, bulk status, and appointment list together on the first tab.
- Moved **Print Confirmed / Reconfirmed Patients List** to the second tab.
- Reorganized Manual Booking so appointment details remain at the top and patient selection uses two tabs: **Existing Patient** and **Create New Patient**.
- Added a Book Appointment button inside each patient tab.
- Removed the MyCareBD Push Notifications menu, Firebase settings screen, direct mobile push hooks, and related runtime feature files.
- SMS and email appointment notifications remain available through their existing settings.
