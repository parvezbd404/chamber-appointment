# Chamber Appointment System v1.0.66

## Reconfirmed appointment lock

- Once a chamber manager sets an appointment status to **Reconfirmed**, the patient can no longer modify or cancel that appointment from the website or MyCareBD mobile API.
- Patients see a clear instruction to call the chamber manager for any date change or cancellation.
- Chamber managers retain their existing backend ability to edit or cancel reconfirmed appointments.
