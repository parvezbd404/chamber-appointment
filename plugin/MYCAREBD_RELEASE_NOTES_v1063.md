# Chamber Appointment System v1.0.63

## Date-filtered appointment worklist: keep patient details open

- When the **Appointments** page is filtered by a date, all appointment row details remain expanded.
- The WordPress down-arrow / row-details toggle is hidden for that filtered daily worklist.
- Chamber managers can view patient name, mobile, doctor, status, reconfirm-call status, serial and all other columns without opening each row.
- The change applies only to an operational date-filtered worklist; ordinary unfiltered appointment lists retain WordPress default behavior.
