# MyCareBD v1.0.55 — Direct FCM Push

- Adds direct Firebase Cloud Messaging (FCM V1) support for the MyCareBD Android Studio APK.
- Native Android FCM tokens can be registered through the existing secure mobile API.
- Adds **Chamber → MyCareBD Push Notifications** settings for FCM enablement, Firebase project ID, service-account JSON, and an authorized test notification.
- Keeps the permanent in-app alert inbox and legacy Expo-token delivery for previous app versions.
- Supports appointment confirmations/status updates and Diabetes Care enrollment, treatment-plan, due, and overdue reminders.
